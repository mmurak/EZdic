プログラムの概要

　「EZdic」は、日英社の「熟語本位 英和中辭典（再訂版）」（斎藤秀三郎著）をパソコンのブラウザから閲覧したり、検索（見出し語のみ、例文は検索できません）する目的で開発したプログラムです。

　辞書データ（画像）自体は「国立国会図書館デジタルコレクション」（NDLDC）で公開されている昭和5年に印刷された版を使用しているため、最新版をお望みの方は岩波書店様が販売している「熟語本位　英和中辞典　新版」をお勧めします（なお、本プログラムは「新版」には対応しておりません）。

必要なもの

  • 空き容量が400MB以上あるPC（インストール時にはNDLDCからの圧縮ファイルを読み込み、展開することになるため、この倍ほど必要になります）。
  • このプログラム（ウェブアプリのため、ブラウザからローカルファイルを閲覧するかたちになります）。
  • NDLDCで公開されている「熟語本位 英和中辭典（再訂版）」の画像データ

インストール方法

  1. EZdic-master.zipをPC環境上の適当な場所にダウンロードし、解凍する。
  2. NDLDCのサイトから画像データをダウンロードする。
    1. 閲覧ページhttps://dl.ndl.go.jp/pid/1679475/1/1にアクセスする。
    2. ページ最下部までスクロールする
    3. ［ダウンロード］セクションのファイル形式を「JPG」、解像度を「通常の解像度（軽量）」、範囲を「すべてのコマ」に設定し、［ダウンロード］ボタンをクリックする。
    4. 数分経つと、ZIP形式のファイルがダウンロードされるため、それを解凍する。
  3. 解凍によって「Saito's idiomological English-Japanese dictionary = 熟語本位英和中辭典_noBundleName」という名前のフォルダ（ディレクトリ）が作成され、その中に810コマの画像ファイルが展開される。このフォルダごと手順「1」で解凍したフォルダ内にある「resources」フォルダ内へと移動する。

使い方

プログラムの起動

　インストール方法の「1」で解凍したフォルダ内にある「index.html」をダブルクリックまたはオープンすることで、プログラムが起動します。ブックマークしておくと次回以降の起動が便利になります。

検索の実行

　検索語入力エリアに半角アルファベットを入力した後、［enter]キーを押すことで検索処理が実行されます。また、［esc］キーを押すことで検索語入力エリアがクリアされ、入力準備が整います。

ページ拡大機能

　ページ画像が表示されている領域でマウスをクリックすると、ページ拡大機能が有効化／無効化できます。

ページの読み進め／戻り

　［↑］キーを押すと1ページ前に、［↓］キーを押すと1ページ後に移動します。

履歴

　［履歴］ドロップボックスから過去の履歴を再検索することができます（最大50語まで）。

栞（しおり）紐

　［Shift］キーを押しながら［栞］ボタンをクリックすると、現在表示されているページに栞を設定することができます（設定できるのは1つだけです）。［Shift］キーを押さずに［栞］ボタンを押すと、以前に設定したページが表示されます。

「熟語本位 英和中辭典」の歴史と各版について

　「熟語本位 英和中辭典」には以下の版が存在しています。

正則英語學校出版部（斎藤秀三郎氏）が制作し、日英社が出版したもの

  • [A]. 第1版：1915年（大正4年）7月20日発行
  • [B]. 改訂版：1917年（大正6年）8月31日発行
  • [C]. 再訂版：1919年（大正8年）10月10日発行

豊田實氏の増補で、岩波書店が出版したもの

  • [D]. 增補新版：1936年（昭和11年）3月15日発行
  • [E]. 新増補版：1952年（昭和27年）発行
  • [F]. 新版：2016年（平成28年）10月25日発行[*1]

[*1] なお、[F]の新版は[E]の新増補版における漢字／かな遣いを変更し、ルビの追加や校注（八木克正氏の手による）を施したものであり、CD-ROM内には[E]と[F]のPDFが収録されています。
[F]については現在も岩波書店から販売されています。https://www.iwanami.co.jp/book/b266364.html
